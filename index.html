ターゲットは車通勤のサラリーマン。通勤時間をメール確認作業に費やし、残業を減らしたい。
そんな人には車内搭載システムを用いた "Moving Work"

　使用方法：再生ボタンを押してください（実際は「エンジンをかける」などがトリガーです）。あらかじめ用意されたメールの送信元・件名が読まれます。本文の要約を読むか聞かれるので、「読んで」or「スキップ」と伝えてください。メールの数（5通）分繰り返されます。※マイクの使用が必要です。

　企画：約0.5日

　制作：約1日

　言語：HTML、CSS、JavaScript、Python

　技術：Text Summarization API、Web Speech API
